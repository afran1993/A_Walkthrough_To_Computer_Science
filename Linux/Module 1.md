# Module 1

Everyday examples of Linux:

| Category | Description |
| :--- | :--- |
| **Airplane** | The in-flight movies are on an embedded Linux system |
| **Internet Access** | The wifi router probably have an embedded copy of Linux running the web management application |
| **TV** | On-screen user interface often runs an embedded version of Linux |
| **Google search** | Almost all of the back office stuff which is used to generate the results of your queries are run on Linux |
| **Websites** | Most likely that website is running on a Linux server |
| **Smart phone** | Most people use an iPhone or an android phone which is based on open source derived from Unix |
| **Tablets (e.g. e-books)** | Most have open source in there somewhere |
| **Windows data recovery disc** | Most are customized Linux distributions |
| **Appliances** | New appliances with touch screen operation runs on Linux |
| **Self Driving Cars** | Running Linux. |
